# LeafApi

## 技术栈

- 前端：ant design pro
- 后端：springBoot,springCloud gateway,MybaitsPlus,Hutool,dubbo+nacos

## 功能模块

### 管理员模块

 - 接口管理（增删改查）
 - 接口分析（统计接口调用次数）
 - 上下线接口

### 用户模块

- 使用SDK调用接口
 - 查看接口
 - 在线调用接口
